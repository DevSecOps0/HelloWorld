왕초보를 위한 깃허브 사용법

git status	저장소 상태
git add	파일이나 폴더를 인덱스에 추가	git add [file_pattern]
git commit	인덱스에 추가된 파일이나 폴더의 내용을 저장소에 쓸 때	git commit -am "A first commit"
git branch	브랜치에 다양한  작업
git branch [branch-name]	브랜치 만들기
git branch	브랜치 목록보기
git branch -d [branch-name]	지정한 브랜치를 삭제
git checkout	로컬 저장소의 브랜치를 전환할 때 사용	git checkout [branch-name]
git log	로컬 저장소의 커밋 히스토리를 탐색	git log -n 10
git grep	저장소의 파일 내용에서 검삭하고자 할 때	git grep "검색단어"
git clone	기존 원격저장소를 로컬에 다운로드 하기 위한 명령어	git clone [url]
git remote	원격 저장소를 조작하는데 사용
git remote	원격 저장소의 이름 목록을 표시
git remote -v	원격 저장소에 대한 자세한 목록보기
git remote add [name] [url]	원격 저장소를 추가
git remote rm [name]	원격 저장소를 제거
git reset	로컬 저장소의 커밋을 취소하기 위한 명령	git reset -soft HEAD ^
git merge	현재 브랜치에 다른 지점에서 변경사항을 병헙	git checkout master git merge big-fix
git pull	원격 브랜치의 변경사항을 캡쳐하기 위해 사용	git checkout master git pull origin master
